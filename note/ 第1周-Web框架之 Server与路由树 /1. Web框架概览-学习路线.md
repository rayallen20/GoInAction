- Web框架概览

看不同的框架是怎么设计的

- 路由匹配

	课上会讲3种:

	- 静态路由匹配
	- 通配符匹配
	- 参数路径

	作业:
	
	- 正则匹配

- AOP方案

	通过AOP方案实现可观测性Middleware(tracing、metrics、log)
	
以上是一个Web框架的核心部分.后续的页面渲染、文件处理与静态资源、Session都可以认为是为了方便使用者而额外引入的功能.因此在进行这3部分的开发时,不会再侵入核心代码.当然,也可以做成侵入式的.
